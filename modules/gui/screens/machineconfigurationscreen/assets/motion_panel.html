Motion

This panel allows you to configure the basic parameters of motion control.

The most important is probably the steps per millimeter paramaters, which allows your machine to move the exact distance you request in each axis


Steps per millimeter

About

When you control your machine, you ( or your control software ) gives it instructions to move to given positions, in millimeters

However, when your board actually controls a given axis, it does so by sending «steps»

You need to know, or calculate, the ratio of steps per millimeter, so that your machine will be capable of moving the correct distance

This calculation depends on your motors, drivers, and drive system ( you can watch this guide on youtube to learn more )

Manual settings

If you already know your values, you can set them manually here

Accuracy

Finding the values by measuring ( with the proportional wizard ) will only be as accurate as your measurements

It is very likely that the values you will find will not be perfectly correct, but just a good approximation

Only calculating the values, either by using the calculator wizard, or manually, will give you the exact right values
