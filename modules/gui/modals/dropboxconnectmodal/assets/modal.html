Connect to Dropbox

Dropbox is an online file sharing service that allows you to store your files on the cloud and to synchronize them across several computers and devices.

If you currently have a Dropbox account, here you can connect this interface to Dropbox. This will allow you to access files stored there directly from this interface, for retrieval, modification and fabrication.

Please click on the botton bellow to connect this interface to Dropbox. You will be taken to the Dropbox website, which will ask you if this interface is allowed to access your accont. After you confirm this, you will be taken back to this interface.

Popups and security

Your browser may prevent this page from opening a window to Dropbox. If that's the case, make sure you follow the correct procedure to allow popups opening for this page, then click on the button again.
